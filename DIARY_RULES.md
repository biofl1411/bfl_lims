# DIARY_RULES.md — 시험일지 계산 규칙 관리대장

> ⚠️ Claude Code는 시험일지 관련 작업 시작 전 반드시 이 파일을 읽고
> 해당 항목의 상태를 확인한 후 아래 규칙에 따라 행동할 것

---

## 📌 상태 코드 정의

| 코드 | 의미 | Claude Code 행동 규칙 |
|------|------|----------------------|
| 🔒 **LOCKED** | 검증 완료 — 수정 절대 금지 | 수정 시도 전 사용자에게 명시적 허락 요청. 허락 없으면 건드리지 않음 |
| ✅ **VERIFIED** | 정상 동작 확인됨 | 관련 작업 시 "지금도 잘 되나요?" 먼저 질문 후 진행 |
| 🔧 **WIP** | 작업 진행 중 | 자유롭게 수정 가능. 작업 완료 후 반드시 사용자에게 상태 업데이트 요청 |
| ❓ **UNKNOWN** | 검증 미완료 | 작업 전 사용자에게 현재 동작 여부 확인 필요 |

---

## 🔧 작업 완료 후 필수 질문 (WIP → 상태 업그레이드)

작업을 마쳤을 때 Claude Code는 반드시 아래를 사용자에게 물을 것:

```
방금 작업한 [항목명 / 식품유형]의 [수정범위] 계산이 잘 되고 있나요?

① 잘 됨, 더 개선 여지 있음   → ✅ VERIFIED 로 업데이트
② 완벽함, 절대 건드리지 말 것 → 🔒 LOCKED 로 업데이트
③ 아직 확인 중               → 🔧 WIP 유지
④ 문제 있음                  → 원인 분석 후 재작업
```

## 📋 항목별 규칙 목록

---

### [납 / 전체]

- **항목명**: 납 (Lead, Pb)
- **식품유형**: 전체 (식품유형별 기준치 다름 — 아래 세부 참고)
- **수정범위**: calculations (환산 공식), judgmentRule (기준치 비교)
- **상태**: 🔧 WIP
- **작업 시작일**: —
- **검증일**: —
- **검증방법**: —
- **메모**:
  - 식품유형마다 기준치(mg/kg)가 다름 → 식품유형별 분리 관리 필요
  - 단위 환산(mg/kg ↔ ppm) 계산식 개선 중
  - 식육 / 음료수 / 영아용식품 등 기준치 별도 적용 예정

---

### [금속성이물 / 전체]

- **항목명**: 금속성이물
- **식품유형**: 전체
- **수정범위**: calculations (검출 크기 환산), judgmentRule (mm 기준 비교)
- **상태**: 🔒 LOCKED
- **작업 시작일**: 2026-04-03
- **검증일**: 2026-04-03
- **검증방법**: 회화 후(g) 70.9071, 자석봉 회화 전(g) 70.9034, 검체 채취량(g) 503.31 → 원시값 7.351334, 규칙적용 7.35 mg/kg, 판정: 적합 (기준 10.0 미만)
- **메모**:
  - 계산식, 판정, 인쇄 폼 모두 정상 확인
  - 수정 절대 금지 — 사용자 명시적 허락 없이 건드리지 않음

---

### [세균수 / 전체]

- **항목명**: 세균수
- **식품유형**: 전체 (냉동식품, 즉석섭취식품, 음료류 등 기준 상이)
- **수정범위**: calculations (CFU/g 계산), judgmentRule (기준치 지수 비교)
- **상태**: 🔧 WIP
- **작업 시작일**: —
- **검증일**: —
- **검증방법**: —
- **메모**:
  - CFU/g, CFU/mL 단위 혼용 — 시료 상태에 따라 구분
  - 기준치가 지수(10^5 등) 형태 — 일반 숫자 비교 불가, 지수 비교 로직 필요
  - 냉동즉석섭취식품 / 비가열 / 가열 후 섭취 등 세부 분류별 기준 다름

## 📌 수정범위 코드 정의

| 코드 | 설명 |
|------|------|
| `calculations` | 계산식, 환산 공식, 중간값 처리 |
| `judgmentRule` | 적합/부적합 판정 기준치 비교 로직 |
| `inputFields` | 입력 필드 추가/변경 (표준품 농도, 장비번호, 희석배수 등) |
| `ui` | 화면 레이아웃, 표시 방식 (전체 공통 적용) |
| `diaryContent` | 시험일지 본문 텍스트 양식 |
