안녕하세요.

식품의약품안전처 통합실험실정보관리시스템(LIMS) 대리 이누리입니다.

 

통합LIMS WEB API 신규 기관 생성을 위한 API 개발 가이드를 전달드립니다.

 

인증서는 아래 두 종류가 포함되어 있습니다.

1.    기관인증서 ((주)바이오푸드랩 )  -  O000026.jks
2. 시스템운영기관 인증서(테스트용) - O000170.jks

API 개발 및 구축 테스트 시에는 시스템 운영기관 인증서(O000170) 를 사용하여 진행해주시고,
운영 시스템에 적용하실 때에는 기관 인증서(O000026) 를 사용해주시기 바랍니다.

 

테스트용 시스템 운영기관(O000170)의 계정은 전달주신 메일 주소를 기준으로 생성되었으며, 

해당 계정의 비밀번호가 메일로 전달되었습니다. 

 

 

또한, API 시나리오별로 테스트가 가능한 페이지도 제공하고 있습니다.
테스트 페이지 경로: https://wslims.mfds.go.kr/ApiClient/



임시비밀번호 : ODSxcBii
로그인 후 마이페이지에서 비밀번호를 변경해 주시기 바랍니다.
아이디는 엑셀 파일에 있습니다ㅣ.
 

 

※ API를 개발하거나 사용하는 PC 또는 서버의 MAC 주소를 시스템에 등록해야 해당 인증서를 통해 정상적인 접근이 가능합니다.
MAC 주소를 본 메일에 회신해주시면 등록 처리해드리겠습니다.
