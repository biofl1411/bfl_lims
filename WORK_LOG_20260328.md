## 영업관리 (salesMgmt.html)

### 1. 비교견적 총계 행 개선
- 차액 옆에 **경쟁사 소계 / BFL 소계 / 총계(BFL-경쟁사)** 레이블 표시
- 총계 색상: BFL이 저렴하면 초록, 비싸면 빨강
- 검사목적 재선택 시 선택 안 되는 버그 수정 (`oninput` + `onchange` 이중 이벤트)

### 2. 익일 계획 신규 영업 전면 재구성
- 검색 조건 UI: 인허가분야 / 시도 / 시군구 / 읍면동 4개 드롭다운
- **전체보기** / **신규업체 알림** 2탭 + 신규 건수 뱃지
- 시도→시군구→읍면동 연쇄 필터 적용

### 3. 제품보기 버튼
- FSS 카드에 "+ 추가" 아래 **제품보기** 버튼 추가
- `fss_products` 쿼리로 품목명·품목유형·허가일 토글 패널 표시

### 4. 추가 버튼 실제 구현
- `addAreaCompanyToTmr` — 업체를 익일 신규 영업 목록에 추가, 업종 메모 자동 입력, 중복 방지

### 5. 업무일지 목록 개선
- 방문 업체 칩 **세로 → 가로 정렬** (nowrap)
- 기존 업체 수 / 신규 영업 수 / 익일 계획(기존 N · 신규 N) 표시

### 6. 신규영업 활동 Section 4 개선
- 이전 영업일지의 익일 신규 영업 계획을 불러와 결과 입력란 제공
- 결과 입력: **만남 · 명함 · 호의적 · 견적 · 소개서** 5개 토글 버튼 + 메모
- 이전 일지 탐색: 날짜 역산 대신 Firestore 쿼리로 `tomorrowNew` 있는 **가장 최근 일지** 자동 탐색
- 이전 계획 없을 때 안내 문구 표시

### 7. 업무일지 날짜 선택
- 목록 헤더에 **날짜 피커** 추가 → 특정 날짜 일지 바로 열기 (테스트 용도 포함)

### 8. 센터 요약 블록
- 업무일지 목록 위에 **센터 요약** 추가
  - 로그인 사용자 이름 선택 드롭다운 (Firestore `users` 동적 로딩, localStorage 저장)
  - 선택한 사용자의 부서를 동적 조회 → 같은 부서 소속 전원 표시
  - 인원별: 이름 / 월 매출 / 최신 일지 날짜
  - 부서명 하드코딩 없음

## 접수 관리 > 접수 현황 (receiptStatus.html)

### 9. 날짜 필터 버튼 추가
- **전일 · 1주일 · 2년 · 3년** 버튼 추가
- 순서: 전일 · 오늘 · 1주일 · 이번달 · 3개월 · 6개월 · 9개월 · 12개월 · 2년 · 3년 · 전체

### 10. 필터 3열 재배치
- **1열**: 날짜 범위 + 날짜 버튼
- **2열**: 전체 상태 · 전체 분야 · 전체 긴급 · 전체 신호등 · 전체 담당자 · 초기화 · 검색
- 우측: 총 N건 · 새로고침 · 엑셀 · 컬럼

### 11. 태그 기반 검색
- 검색창 입력 후 **Enter** → 파란 라벨 태그 생성
- 태그 여러 개 **AND 조건** 적용
- 빈 입력창에서 **Backspace** → 마지막 태그 삭제
- 태그 × 버튼으로 개별 삭제 / 초기화 버튼으로 전체 초기화
- **줄임 단어 인식**: `자가` → 자가품질위탁검사용 등 alias 맵 적용
- 검색 필드: 접수번호·거래처·제품명·검사목적·주소·담당자·대표자·성적서번호 등 전체 필드

### 12. 부분 매칭 (Fuzzy)
- 2글자 n-gram 매칭: `파스타` 입력 시 `스타` 공통으로 `피스타치오` 포함 항목도 매칭
- 완전 포함 우선, 부분 일치 폴백
