```javascript
var toRemove = [];
for (var r = minR; r <= maxR; r++) {
  for (var c = minC; c <= maxC; c++) {
    if (r === minR && c === minC) continue;
    var cell = _tmGetCellAtCol(allRows[r], c);
    if (cell) toRemove.push(cell);
  }

toRemove.forEach(function(c) { if (c.parentElement) c.parentElement.removeChild(c); });
```

## 4. 결과 입력 일지 기반 상세 입력폼 미표시 버그 수정 (`testResultInput.html`)

### 증상
세균수는 **▼ 접기 / 상세 입력 클릭** 버튼이 표시되는데,
납(Pb) 등 이화학 항목은 단순 텍스트 "입력" 필드만 표시되고 폼이 안 열림

### 원인
Firestore에서 시험항목이 `"납||자기품질위탁검사용"` 형태로 저장되어 있음.
`findFormTemplate(item.testItemName, ...)` 호출 시 `"납||자기품질위탁검사용"` 으로 검색하지만,
템플릿은 `representativeItem = "납"` 으로 저장되어 있어 **이름 불일치 → 템플릿 미탐색**

```
"납||자기품질위탁검사용"  ≠  "납"
→ findFormTemplate() 반환값 null
→ toggleBtn = '' (버튼 없음)
→ inputPlaceholder = '입력' (단순 텍스트 필드만 표시)
```

### 수정
`findFormTemplate` / `findFormTemplateOverrides` 함수 내부에서 `||` 이후 분리

```javascript
// 수정 전
var name = itemName.trim();

// 수정 후
var name = itemName.split('||')[0].trim(); // "납||자기품질위탁검사용" → "납"
```

→ 납 포함 모든 이화학 항목에서 일지 기반 상세 입력폼 정상 표시

커밋: `abb7a89`
